# Time Tracking Dashboard ⏱️

A modern, interactive **time-tracking dashboard** — rebuilt as a showcase by **[Governor Studio](https://governor-studio.vercel.app)**.

Track hours across activities and switch between daily / weekly / monthly views with animated figures.

## Features
- 📊 Six activity cards with current vs. previous-period hours
- 🔁 Daily / weekly / monthly toggle — figures **count-animate** on change
- 🎨 Color-coded categories with icons
- 📱 Responsive layout, accessible, reduced-motion aware

## Stack
**Vite · React 19 · Tailwind CSS v4 · Framer Motion**

## Run locally
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
```
